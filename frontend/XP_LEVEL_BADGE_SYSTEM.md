### 1-5. XP 배율 시스템

최종 XP = `게임 XP × 스트릭 배율 × 버프 배율`

#### 스트릭 배율 (연속 접속)
| 연속 일수 | 배율 |
|-----------|------|
| 0~2일 | 1.0× |
| 3~6일 | 1.2× |
| 7일 이상 | 1.5× |

> 코드 위치: `App.jsx:120-127`

#### 임시 2배 버프
- 특정 이벤트/조건에서 발동
- 만료 시간: localStorage `xp_buff_expires` (타임스탬프)
- 배율: 2.0×

> 코드 위치: `App.jsx:142-144`

## 5. XP 획득 전체 흐름

```
게임 클리어 / 정답
    ↓
onHanjaAcquired(id, xp)  ← App.jsx에서 전달
    ↓
handleHanjaAcquired(id, xpAmount)  [App.jsx:172-185]
    1. 스트릭 배율 계산 (1.0x / 1.2x / 1.5x)
    2. 버프 확인 (2.0x)
    3. finalXp = Math.round(xpAmount × streakMult × buffMult)
    4. setUserXp(prev => prev + finalXp)
    ↓
updateMissionProgress(type, 1)  [useDailyMission.js]
    - 해당 게임 미션 진행도 +1
    - 완료 시 onBonusXp(missionXp) 콜백
    ↓
(모든 미션 완료 시)
    addBonusXp(50)  [App.jsx:169]
    CharacterToast 표시
```
